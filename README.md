# 🚀 Git & GitHub Practice

## 📖 Про проект
Це мій тренувальний репозиторій, який я створила для опанування системи контролю версій Git та платформи GitHub. Тут я на практиці тестую консольні команди та робочі процеси розробників.

## 🛠 Що я тут навчилася робити:
* Створювати локальний репозиторій (`git init`).
* Робити знімки коду та читати історію (`git log`, `git status`).
* Приховувати "сміттєві" та скомпільовані файли за допомогою `.gitignore`.
* Відправляти код на віддалений сервер (`git push`).
* Працювати з паралельними гілками (`git branch`, `git checkout`/`switch`).
* Створювати та успішно мерджити **Pull Requests**.
